# Lions Auto Protection – Junior Software Developer Bootcamp (Beginner)

## Overview
This bootcamp is a full, beginner-friendly curriculum designed to take a learner from **zero experience** to **job‑ready** for the Lions Auto Protection Junior Software Developer role. It is structured around the exact technology stack and responsibilities from the job posting: **C#/ASP.NET Core/EF Core/MSSQL**, **HTML/JavaScript/CSS (Tailwind)**, **GitHub**, and **WordPress**. The plan is comprehensive and all‑inclusive, with structured lessons, labs, weekly projects, and a capstone built to simulate real business workflows.

**Format:** 16 instructional weeks (4 months), plus an optional Week 0 setup module, ~15–20 hours/week (can be scaled)

**Outcomes:** By the end, learners can:
- Build and maintain ASP.NET Core web apps with EF Core and MSSQL
- Create responsive UI with HTML/CSS/JS and Tailwind
- Use Git/GitHub in a team workflow
- Debug, test, and document code
- Collaborate with stakeholders and deliver features on time
- Support WordPress basics and integration needs

---

## Phase 0 – Bootcamp Setup (Week 0)
**Goal:** Prepare the learner with tooling, workflows, and study habits.

### Topics
- How to learn programming effectively (growth mindset, problem solving)
- Development environment setup (Windows recommended for .NET; macOS/Linux ok)
- Tools: Visual Studio Community, VS Code, SQL Server Express or Developer Edition, Git, GitHub Desktop
- Understanding the job role and expectations

### Deliverables
- All tools installed and verified
- GitHub account set up
- Personal learning journal created

## Phase 1 – Programming Fundamentals with C# (Weeks 1–3)
**Goal:** Build programming foundations with C# and basic problem-solving.

### Week 1 – Core Concepts
- Variables, data types, operators
- Input/output
- Control flow (if/else, switch)
- Loops (for, while)

**Practice Lab:** 30+ small exercises (calculator, temperature converter, loops)

### Week 2 – Functions and Data Structures
- Methods/functions
- Arrays and lists
- Basic string manipulation
- Exception handling (try/catch)

**Mini Project:** Console-based inventory system

### Week 3 – Object-Oriented Programming (OOP)
- Classes, objects, constructors
- Encapsulation, inheritance, polymorphism
- Interfaces

**Mini Project:** Library management system (console)

**Goal:** Learn how the web works and build basic UI.
