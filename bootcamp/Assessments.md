# Bootcamp Assessments and Checkpoints

## Weekly Checkpoints
Each Friday includes a mini-project or review. Students should demonstrate:
- Correctness and clean code
- Clear explanations of decisions
- Basic troubleshooting

## Major Assessments
- Week 5: Web foundations mini-project
- Week 9: EF Core + MSSQL project
- Week 13: Requirements and documentation package
- Week 16: Capstone delivery and demo

## Rubric (High Level)
- **Functionality (40%)**: features work as specified
- **Code Quality (20%)**: readability, structure, naming
- **Testing & Debugging (15%)**: evidence of tests and fixes
- **Documentation (15%)**: README and usage notes
- **Professionalism (10%)**: communication and collaboration
