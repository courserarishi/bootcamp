# Week 01 – C# Fundamentals

## Day 01: C# syntax, variables, and types

### Lesson Content
- C# program structure: namespaces, Main method, statements.
- Value types vs reference types; common primitives.
- Console input/output and string interpolation.

### Instructor Demo
- Build a console calculator with +, -, *, /.
- Create a temperature converter (C↔F).

### Lab / Workshop
- Write a BMI calculator that reads input and prints results.
- Add basic validation for numeric input.

### Assessment / Checkpoint
- Submit calculator and BMI app with sample output.

## Day 02: Control flow

### Lesson Content
- if/else, else if, and switch statements.
- Boolean logic with &&, ||, !.
- Using guard clauses to simplify logic.

### Instructor Demo
- Create a grading program (A–F) using if/else.
- Refactor to switch for menu selection.

### Lab / Workshop
- Build a discount calculator with tiered pricing.
- Create a day-of-week scheduler using switch.

### Assessment / Checkpoint
- Submit discount calculator with test cases.

## Day 03: Loops

### Lesson Content
- for, while, do-while loops and their use cases.
- Loop counters, accumulators, and break/continue.
- Avoiding infinite loops and off-by-one errors.

### Instructor Demo
- Build a number guessing game.
- Generate a multiplication table.

### Lab / Workshop
- Sum a list of numbers entered by the user.
- Print even numbers from 1 to 100.

### Assessment / Checkpoint
- Submit guessing game with at least 3 rounds.

## Day 04: Debugging and clean code

### Lesson Content
- Using breakpoints, step-in/step-over/step-out.
- Reading stack traces and fixing common errors.
- Naming conventions and formatting.

### Instructor Demo
- Debug a broken calculator app.
- Refactor variables and extract methods.

### Lab / Workshop
- Fix 3 bugs in a provided sample app.
- Refactor the app to improve readability.

### Assessment / Checkpoint
- Submit bug-fix notes and before/after screenshots.

## Day 05: Mini-lab review

### Lesson Content

- Documenting console app usage.

### Instructor Demo
