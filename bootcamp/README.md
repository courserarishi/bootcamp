# Lions Auto Protection Bootcamp (Full Daily Plan)

This directory contains the **full daily curriculum** for the Junior Software Developer bootcamp. It expands the high-level plan in `BOOTCAMP_PLAN.md` into week-by-week folders with **daily learning objectives, lessons, exercises, and deliverables**.

## Structure
```
bootcamp/
  Week-00-Setup/
  Week-01-CSharp-Fundamentals/
  Week-02-CSharp-Data-Structures/
  Week-03-OOP/
  Week-04-Web-Foundations/
  Week-05-JavaScript/
  Week-06-Tailwind/
  Week-07-Git-GitHub/
  Week-08-ASPNET-MVC/
  Week-09-EFCore-MSSQL/
  Week-10-Auth-Identity/
  Week-11-Testing-Debugging/
  Week-12-WordPress/
  Week-13-Collaboration-Delivery/
  Week-14-Capstone-Planning/
  Week-15-Capstone-Build/
  Week-16-Capstone-Polish/
```

Each week has **Day-01** through **Day-05** markdown files with:
- Objectives
- Lessons
- Guided practice
- Deliverables
- Homework/reading

## Additional Docs
- `Assessments.md` for checkpoints and grading rubric
- `Capstone-Brief.md` for the final project requirements
- `course-content/` for full lesson content by week

Use this as a complete, beginner‑friendly program from **zero experience** to **job‑ready**.
